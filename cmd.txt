
Install angular
  https://www.baeldung.com/linux/multiple-node-js-versions
  nvm use 14
  npm install -g @angular/cli@15

Install libs to read and parse yaml files:
  npm install yaml

Start local server:
  nvm use 14
  ng serve

Build and deploy:
  nvm use 14
  ng build --prod --base-href ./
  nvm use 20
  npm install -g firebase-tools
  firebase deploy
